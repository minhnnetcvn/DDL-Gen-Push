## Installation
### Minimum Requirements
- Node.js version 24.12 LTS or higher

### Install Dependencies
```bash
   cd next-js-forms #Optional if not already in root folder
   npm i
```

### Protocol Notes
- **Local**: Use HTTP
- **Remote/Public**: Use HTTPS
